Cara Menggunakan Aplikasi

Aplikasi ini membantu Anda mempelajari huruf bahasa isyarat dengan dukungan deteksi tangan menggunakan webcam.

  • Gunakan tombol Simulasi untuk mencoba membentuk huruf dengan tangan.
  • Gunakan tombol Kamus untuk melihat referensi gambar setiap huruf.
  • Untuk mengumpulkan data dan melatih model, gunakan program capture_dataset.py dan train_model.py.
